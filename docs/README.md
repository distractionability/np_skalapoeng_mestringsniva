# Reference Documentation

This folder contains external reference documents used in the score correction analysis.

## Files

### SSB_korrigerte_proveresultater.pdf
**Source:** Statistics Norway (SSB)
**Original title:** "Korrigerte tall for nasjonale prøveresultater over tid"

Official documentation from SSB describing the discovery of calculation errors in national test scores and the correction methodology. This is the primary source for the rescaling parameters used in this analysis.

### UDIR_gjennomforing_resultater.pdf
**Source:** Norwegian Directorate for Education and Training (Udir)
**Original title:** "Gjennomføring og resultater"

Official documentation about national test implementation and results reporting, including competence level (mestringsnivå) definitions and cutoff thresholds.

## Related Data

The official rescaling parameters from SSB Table 1 are stored in `data/rescaling_parameters.csv`.
